Available Color Palettes

The ggsci package offers a collection of ggplot2 color palettes inspired by scientific journals, data visualization libraries, and science fiction TV shows.

For a full list of color palettes included, please refer to the package vignette.

Documentation

The package vignette describes the design and usage of ggsci in detail.

View Package Vignette

Feedback

To contribute new palettes, you are welcome to send pull requests directly. If you love a journal-specific or sci-fi theme color palette that we did not include, please create an issue to inform us. Feel free to send us an e-mail to provide your general feedbacks.
